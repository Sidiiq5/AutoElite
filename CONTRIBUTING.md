# Contributing to AutoElite\n\nThank you for your interest in contributing! This guide will help you get started.\n\n## Getting Started\n\n1. Fork the repository\n2. Clone your fork: `git clone https://github.com/YOUR_USERNAME/AutoElite.git`\n3. Create a feature branch: `git checkout -b feature/your-feature`\n4. Make your changes\n5. Push to your fork: `git push origin feature/your-feature`\n6. Create a Pull Request\n\n## Code Guidelines\n\n- Write clean, semantic HTML\n- Use modern CSS practices\n- Keep JavaScript organized and commented\n- Test on multiple browsers\n- Update README if needed\n\n## Issues\n\nFeel free to open issues for:\n- Bug reports\n- Feature requests\n- Design improvements\n- Documentation\n\n## Pull Requests\n\n- Keep PRs focused and small\n- Write clear commit messages\n- Reference related issues\n- Test your changes\n\nThank you for contributing!\n"
